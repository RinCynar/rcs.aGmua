🌟 Overview

RCS.aGmua is a versatile text encryption tool based on the RC4 encryption algorithm. It allows you to encrypt and decrypt text, manage keys, track history, and more.

⚙️ Features:

  • 🔒 Encrypt and Decrypt Text
  • 🔑 Key Management
  • 📜 History Management
  • 🔄 Update Notifications
  • 🔧 Configuration Reset
  • 💥 Brute-force Decryption
  • 💻 User-friendly Commands

📋 Installation Steps

  1. Ensure Python is installed.
  2. Install required modules: pip install arc4 requests

📥 Usage

python aGmua.py -i

📜 Command List:

  • relp: Display usage instructions
  • rak : Add a new key
  • rch: Clear encryption/decryption history
  • rck: Display saved encryption keys
  • rdk : Delete a specified key
  • rxit: Exit the tool
  • rsh: Show encryption/decryption history
  • rc : Perform brute-force decryption
  • res: Reset to default configuration
  • rcu: Check for updates

💡 Example:

Encrypting Text:

python aGmua.py -i

Enter text to encrypt:

 # Hello, World!

Decrypting Text:

python aGmua.py -i

Enter encrypted text:

 - 5D41402ABC4B2A76B9719D911017C592
